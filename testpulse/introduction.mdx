# What is TestPulse?

TestPulse is an **Azure DevOps extension** that lives directly inside Test Plans. It turns your test execution data into professional reports — PDF, HTML, Word, and Excel — without any external tools, backend servers, or copy-pasting.

## What it does

- **Fetches** test results, suites, work items and attachments via the ADO REST API
- **Computes** KPIs: pass rate, execution rate, flaky detection, traceability coverage
- **Exports** to PDF, HTML, Word (.docx), and Excel (.xlsx) with a single click
- **No data leaves** your Azure DevOps instance — everything runs in the browser

## When to use TestPulse

| Use case | Recommended format |
| --- | --- |
| Formal QA delivery to a client | PDF with signature page |
| Sprint review dashboard | HTML (interactive, shareable) |
| Audit trail (ISO, SOC2) | PDF \+ Excel |
| Stakeholder executive summary | PDF with charts |
| Raw data analysis | Excel (4 sheets: KPIs, Results, Traceability, Bugs) |

## What it is NOT

- **Not a test runner** — TestPulse reads existing results, it doesn't execute tests
- **Not a cloud service** — no data leaves your Azure DevOps instance
- **Not a replacement for ADO Test Plans** — it complements it with reporting

## Supported targets
